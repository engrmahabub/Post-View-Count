== Description ==
The SM Post View Counter is a wordpress widget plugin used to show post view count for.

= Features =
* View Total and Todays view count

== Installation ==
1. Upload the entire `sm-post-view-count` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now SM Post View Count widget abailable in your widget section.

= Usage =

1. After active the plugin you can see total and todays view count
